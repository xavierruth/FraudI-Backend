## Como rodar localmente
1. Crie e ative um ambiente virtual
    ```perl
        python -m venv venv
        .\venv\Scripts\activate  # Windows
        source venv/bin/activate  # Mac/Linux
    ```
2. Instale as dependências
      ```perl
        pip install -r requirements.txt
    ```
3. Incie a aplicação 
    ```perl
        python app.py
    ```
